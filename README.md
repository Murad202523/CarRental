🚗 CarRental (React + Tailwind)

📌 Description
A car rental UI web application built using React and Tailwind CSS.

🚀 Features
- Car listing interface  
- Booking UI  
- Responsive layout  

🛠 Tech Stack
- React  
- JavaScript (ES6+)  
- Tailwind CSS  

▶️ How to Run
1. Clone the repository  
2. Run:
npm install  
npm run dev
